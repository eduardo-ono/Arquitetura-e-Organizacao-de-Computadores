Concorrência e Paralelismo > Lei de Amdahl

Prof. Eduardo Ono


Lei de Amdahl:

$$ s = { 1 \over (1-p + {p \over N}) } $$

onde:
s = speedup (ganho de desempenho)
p = fração do código paralelizável
N = número de unidades de processamento

A Lei de Amdahl é também comumente utilizada para se determinar a fração do código paralelizável. Assim sendo, a função pode ser escrita em termos de p:

$$ p = { { { 1 \over s } - 1 } \over { { 1 \over N } - 1 } } $$
